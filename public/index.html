🔌 Connecting an Account

  • TopstepX: Log in at topstepx.com → Settings → API Keys → generate a key. Enter your username + API key here.
  • Lucid Markets: Obtain your API key & secret from the Lucid portal. Enter your Account ID (e.g. ACC123456).
  • Bitunix: Log in at bitunix.com → Account → API Management → create a key with trading permissions. Enter your API Key and Secret Key here.
  • After connecting, click Test to verify credentials reach the broker live.

📋 Step-by-Step Setup Guide
1
Create or select a webhook
Pick an existing webhook below to see its URL, or click + New Webhook above to create one first.
Your Webhook URL
2
Open TradingView and create an Alert
On your chart, click the Alerts (clock 🕐) icon in the right toolbar, then press + Create Alert.
3
Set your alert condition
Under the Settings tab, choose your indicator, strategy signal, or price level as the condition.
4
Paste the Webhook URL
Switch to the Notifications tab inside the alert dialog. Enable Webhook URL and paste the URL from Step 1.
5
Set the alert Message (JSON)
In the Message box, paste this JSON — TradingView fills in the dynamic fields automatically:
{{ "action": "{{strategy.order.action}}", "ticker": "{{ticker}}", "qty": {{strategy.order.contracts}}, "signal": "my-strategy" }}
For manual (indicator) alerts use "action":"buy" or "action":"sell" directly.
6
Save and you're live! 🎉
Click Save in TradingView. When the alert fires, the order routes instantly to your broker. Watch it appear in Trade Log.
⚠️ Tips
  • TopstepX contract IDs look like CON.F.US.ENQ.U25 — use this as the ticker value.
  • Add "sl" / "tp" for price-based brackets, or "slTicks" / "tpTicks" for tick-based brackets.
  • The signal field is logged in the Trade Log — great for labelling strategies.
  • Make sure your account is connected and the Test Connection passes before going live.

Order Flow Auto-Trader
AI-driven order flow engine — Delta · Imbalance · Absorption · EMA Trend · ATR Regime · 2:1 R:R
⚡ How Order Flow Auto-Trading Works
4-metric voting engine scores every bar across Delta strength, Cumulative Delta trend, Volume Imbalance, and Absorption events. A bonus is added when the EMA slope agrees.
ATR regime filter suppresses signals when volatility is too low (choppy, no follow-through) or too high (gapping past your SL). Only normal regime bars trade.
2:1 Risk-Reward — TP is always SL × R:R ratio. At a 40% win rate a 2:1 system breaks even; above 40% is profitable. Edge comes from only trading high-confidence, trend-aligned setups.
Circuit breakers — max daily trades, max daily loss, and consecutive-loss limit automatically halt the bot to protect your account.
